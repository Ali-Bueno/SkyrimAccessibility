## Mod Organizer 2
### How to Install Mod Organizer 2
1. Download Portable [Mod.Organizer-2.5.2.7z](https://github.com/ModOrganizer2/modorganizer/releases/tag/v2.5.2) and extract it where you like.
2. Make a shortcut for ModOrganizer.exe and move it to desktop so you don't have to launch it from MO2 folder every time.
3. Launch MO2
4. Create a new Instance.  
5. Select Create Portable Instance.  
6. Browse for a game to manage. E.G. J:\Games\Skyrim Special Edition  
7. Select game editon.  
8. During Configure your profile settings check to use profile specific ini files and save files.  
9. When selecting folder where data should be stored, just go with the default path, it should be the same folder where MO2 is installed.  
10. Link Mod Organizer to your Nexus Mods account.
11. Finsih Installation.

### How to use Mod Organizer 2
<!-- Update for more details -->
Launch Mod Organizer 2.  
Launch the game at least once.  
Open web browser.  
Download SKSE and install it manually by extracting contents of archive into game folder where .exe files are.  
Download the rest of mods from Nexus Mods using the "Install via Mod Manager" button.  
Download The Skyrim Accessibility Mod.  
Go back to Mod Organizer.  
Use the Tab key to find the Plugins tab control.  
Use the right/left arrows to navigate and open the Downloads tab.  
Use Tab until you reach the tree view with downloads.  
Use the up/down arrows to navigate through the Downloads tab.  
Press Enter to quick-install the selected mod.  
Install all mods in the Downloads tab.  
Use Tab until you reach the tree view with mods.  
Press Shift+F10 to open the context menu and select "Open All Mods" menu.  
Select "Install Mod above".  
Search for the downloaded The Skyrim Accessibility Mod file.  
Press Shift+F10 and select "Send to Highest Priority".  
Press Shift+F10 and select "Enable All Mods".  
Now use Tab to go back to the Downloads tab.  
Use the right/left arrows to navigate to the Plugins tab.  
Use Tab until you reach the tree view with plugins.  
Press Shift+F10 and select "Enable All".  
Use Tab until you reach the combo box to select a program to run, select "SKSE".  
Use the Run button to play the game.
