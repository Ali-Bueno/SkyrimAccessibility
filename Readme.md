# Skyrim Accessibility Mod By Dio Kyrie

## Available Features
- Intro Auto Walkthrough;
- Clairvoyance. A Guiding spell which will lead you to current quest marker via sound trail.
- [Work In Progress] Sound Notify if Walking in Place.
- [Work In Progress] Audio Markers on Items, NPC, Containers, Doors, etc.
- Auto Lockpicking. 25% Chance to open lock.

## Hotkeys
- X Key - [Work In Progress] Lock Camera On Target. List of targets to lock camera on.
- G Key - [Work In Progress] Autowalk. Selet Neraby Target, Interest Target or City.
- V Key - Activate Menu. Interact with NPC, Doors, Containers, etc. Max Distance 10 Meters.
- H Key - None
- U Key - None
- O Key - Toggle Clairvoyance. Turns On or Off Clairvoyance Autocast.
- B Key - [Work In Progress] Character Status. Health, Mana, Stamina, Debuffs.
- K Key - None
- L Key - [Work In Progress] Debug Menu. Write Coordinates and Make ScreenShot.

## Install Guide
1. Install [Skyrim Special Edition](https://store.steampowered.com/app/489830/The_Elder_Scrolls_V_Skyrim_Special_Edition/);
2. Install [Mod Manager](https://github.com/DioKyrie-Git/SkyrimAccessibility/blob/main/Docs/ModManagersGuides.md);
3. Launch Skyrim Launcher with Mod Manager for it to automatically generate config files and close it;
4. Download and Install [Requirements](https://github.com/DioKyrie-Git/SkyrimAccessibility/blob/main/Docs/Requirements.md);
5. Install latest release of [Skyrim Accessibility](https://github.com/DioKyrie-Git/SkyrimAccessibility/releases) mod;
6. Install latest release of [SkyrimTTS](https://github.com/DioKyrie-Git/SkyrimTTS/releases);
- (Optional) Install [Recomended Mods](https://github.com/DioKyrie-Git/SkyrimAccessibility/blob/main/Docs/RecomendedMods.md);
8. Launch SKSE with Mod Manager.
- You can read introduction to Skyrim UI and Controls.
