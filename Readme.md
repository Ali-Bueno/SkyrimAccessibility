# Skyrim Accessibility Mod By Dio Kyrie

## Available Features
- Intro Walkthrough;
- Clairvoyance. A Guiding spell which will lead you to current quest marker via sound trail.
- [Work In Progress] Sound Notify if Walking in Place.
- [Work In Progress] Audio Markers on Items, NPC, Containers, Doors, etc.
- Auto Lockpicking. 25% Chance to open lock. Only available by using Interact Menu.

## Hotkeys
- V Key - Interact Menu. Interact with NPC, Doors, Containers, etc.
- L Key - Help Menu. Lots of situational things and settings.
