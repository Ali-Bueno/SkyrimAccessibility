# The Skyrim Accessibility Mod By Dio Kyrie
## Install Guide
#### 1. Install [Skyrim Special Edition](https://store.steampowered.com/app/489830/The_Elder_Scrolls_V_Skyrim_Special_Edition/);
#### 2. Install [ModOrganizer2](https://github.com/ModOrganizer2/modorganizer/releases);
#### 3. Launch Skyrim Launcher from within MO2;
#### 4. Download and Install Requirements with MO2;
#### 5. Install [The Skyrim Accessibility Mod](https://codeberg.org/DioKyrie/TheSkyrimAccessibilityMod/archive/main.zip) with MO2;
#### 6. Launch SKSE from within MO2.

## Requirements:
#### - [SKSE](https://www.nexusmods.com/skyrimspecialedition/mods/30379?tab=files);
#### - [Addres Library for SKSE plugins](https://www.nexusmods.com/skyrimspecialedition/mods/32444?tab=files);
#### - [Power Of Three's Papyrus Extender](https://www.nexusmods.com/skyrimspecialedition/mods/22854?tab=files);
#### - [UIExtensions](https://www.nexusmods.com/skyrimspecialedition/mods/17561?tab=files);
#### - [Media Keys Fix](https://www.nexusmods.com/skyrimspecialedition/mods/92948?tab=files);
#### - [Better MessageBox Controls](https://www.nexusmods.com/skyrimspecialedition/mods/1428?tab=files);
#### - [Stay At The System Page - AE](https://www.nexusmods.com/skyrimspecialedition/mods/67883?tab=files).

## Available Features
#### - Intro Auto Walkthrough;
#### - Clairvoyance. A Guiding spell which will lead you to current quest marker via sound trail.
#### - Sound Notify if Walking in Place.[Work In Progress]
#### - Audio Markers on Items, NPC, Containers, Doors, etc.[Work In Progress]

## How to use Mod Organizer 2
##### Launch Mod Organizer 2.
##### Open web browser.
#### Download mods from Nexus Mods using the "Install via Mod Manager" button.
##### Download The Skyrim Accessibility Mod.
##### Go back to Mod Organizer.
##### Use the Tab key to find the Plugins tab control.
#### Use the right/left arrows to navigate and open the Downloads tab.
##### Use Tab until you reach the tree view with downloads.
#### Use the up/down arrows to navigate through the #Downloads tab.
##### Press Enter to quick-install the selected mod.
##### Install all mods in the Downloads tab.
##### Use Tab until you reach the tree view with mods.
#### Press Shift+F10 to open the context menu and select "Open All Mods" menu.
##### Select "Install Mod above".
#### Search for the downloaded The Skyrim Accessibility Mod file.
##### Press Shift+F10 and select "Send to Highest Priority".
##### Press Shift+F10 and select "Enable All Mods".
##### Now use Tab to go back to the Downloads tab.
#### Use the right/left arrows to navigate to the Plugins tab.
##### Use Tab until you reach the tree view with plugins.
##### Press Shift+F10 and select "Enable All".
#### Use Tab until you reach the combo box to select a program to run.
##### Make sure "Skyrim Special Edition Launcher" is selected.
##### Use Tab to go to the Run button.
##### Launch the game once and close it.
##### Go back to the combo box and select "SKSE".
##### Use the Run button to play the game.
