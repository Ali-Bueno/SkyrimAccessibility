# Skyrim Accessibility Install Guide for Mod Organizer 2.
1. Install [Skyrim Special Edition](https://store.steampowered.com/app/489830/The_Elder_Scrolls_V_Skyrim_Special_Edition/).
> [!Note]
> Skyrim Anniversary Edition is update with community mods on top of Skyrim Special Edition.
> It is still the same game version, you can use eighther of those two editions.
2. Download latest setup of [Vortex](https://github.com/Nexus-Mods/Vortex/releases) and install it where you like.
3. Launch Vortex Setup.
4. Select folder where to install Vortex and press Install, default foulder should be alright.
5. Launch Vortex.
6. Use Tab and go to Login Button and press Space.
7. Now your browser will open, login to nexusmods if not logged in already.
8. Press Authorise button.
9. Press Continue button.
10. Use Tab and go to Games Button and press Space.
11. Use Tab and go to Table Search For Games and press Space.
12. Type Skyrim Special Edition
13. Use Tab and go to Unmanaged and press Space.
14. Use Tab and go to Launch Button and press Space.
> [!Note]
> You should do this once to generate ini files, you can close the game right after.
15. Open web browser.
16. Download [SKSE](https://www.nexusmods.com/skyrimspecialedition/mods/30379?tab=files) and install it manually by extracting contents of archive into game folder where .exe files are.
17. Download [nvdaControllerClient](https://download.nvaccess.org/releases/stable/) and extract nvdaControllerClient.dll into game folder where .exe files are.
> [!Note]
> If you've done all correctly both skse64_loader.exe and nvdaControllerClient.dll should now be in same folder with SkyrimSE.exe
18. Download all the mods bellow from Nexus Mods using the "Install via Mod Manager" button.
> [!Note]
> - [Addres Library for SKSE plugins](https://www.nexusmods.com/skyrimspecialedition/mods/32444?tab=files);
> - [Power Of Three's Papyrus Extender](https://www.nexusmods.com/skyrimspecialedition/mods/22854?tab=files);
> - [UIExtensions](https://www.nexusmods.com/skyrimspecialedition/mods/17561?tab=files);
> - [Media Keys Fix](https://www.nexusmods.com/skyrimspecialedition/mods/92948?tab=files);
> - [Better MessageBox Controls](https://www.nexusmods.com/skyrimspecialedition/mods/1428?tab=files);
> - [Stay At The System Page - AE](https://www.nexusmods.com/skyrimspecialedition/mods/67883?tab=files).
19. Download latest releases of [Skyrim Accessibility](https://github.com/DioKyrie-Git/SkyrimAccessibility/releases) and [SkyrimTTS](https://github.com/DioKyrie-Git/SkyrimTTS/releases) from GitHub.
20. Use Tab and go to Downloads Button and press Space.
21. Use Tab and go to Clickable Open Folder and press Space.
22. Move both archives of Skyrim Accessibility and SkyrimTTS in this folder.
23. Go back to Vortex.
24. Use Tab and go to Downloads Button and press Space.
25. Use Tab and go to Install Button and press Space.
26. Again Use Tab and go to Install Button and press Space.
27. Use Tab and go to List with 1 Item Dashboard Button and press Space.
28. Use Tab and go to Launch Button and press Space.
